# Bind

- Installation :
  -  `apt install bind9 -y`

Différent fichiers configuration :

- `/etc/named.conf`
- Répertoire `/var/named`
- Fichier de zones `/etc/named.rfc1912.zones`
- Fichier de clé `/etc/named.root.key`
