# REGEX : Expression régulière

Une Regex permet de représentation des modèles de chaînes de caratère.

## Exemple de Regex 

- `^[R|r]egex$` - Cette Regex permet de "match" toutes les chaînes de caractère ayant écrit juste `Regex ou regex`.
